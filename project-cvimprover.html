01 · Problem The Challenge

Job seekers frequently struggle to articulate their professional experience in a way that is both grammatically flawless and impactful enough to pass Applicant Tracking Systems (ATS). While large-scale AI tools exist, they often generate generic "AI-speak" and require massive computational resources. The goal was to build a lightweight, highly deterministic web application that could run efficiently on limited local hardware while providing users with mathematically backed scoring on their CV improvements.

02 · Solution The Approach

I engineered a Flask-based web application integrating the bigscience/bloom-560m language model via Hugging Face and PyTorch. To prevent hallucinatory or "fluffy" outputs, I configured model inference using a strict 15-beam search strategy, early stopping, and a low temperature of 0.15.

To quantify the improvement, I built a custom Heuristic Scoring Engine. This algorithm evaluates text quality by compiling grammar penalties (language_tool_python), Flesch reading ease (textstat), sentiment analysis (DistilBERT), and a predefined CV keyword density check. The frontend then visualizes this through animated dual-ring progress indicators, showcasing the exact score delta between the original draft and the AI-refined version.

03 · Engineering Technical Highlights

Bloom-560m Inference

Optimized a Hugging Face PyTorch pipeline with 15-beam search and low-temperature settings to guarantee professional, deterministic outputs on limited hardware.

Heuristic Scoring

Engineered a multi-layered algorithm combining DistilBERT sentiment analysis, Flesch reading ease, and grammar checks to calculate a definitive CV quality score.

Flask & Vanilla JS

Built a responsive web dashboard with dual-ring animated SVG indicators to visually compare before-and-after scores in real time.

04 · Results The Outcome

The application successfully transforms scattered drafts into professional bullet points while visibly proving its value through heuristic scoring. Currently maintained as a local portfolio project, the architecture is fully prepared for future cloud deployment via platforms like Vercel.
