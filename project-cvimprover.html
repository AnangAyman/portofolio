DeepSeek AI CV Improver

An intelligent resume optimization engine leveraging DeepSeek R1 and LangChain to reconstruct unformatted text into ATS-friendly, high-impact bullet points.

Solo Developer

Timeline

Domain

Core Tech

The Challenge

Job seekers often struggle to quantify their achievements and format their experience in a way that passes Applicant Tracking Systems (ATS). While many AI resume builders exist, they frequently generate overly repetitive, detectable "AI-speak" or require expensive subscriptions. I wanted to build a free, lightweight tool that specifically targets the most difficult part of resume writing: crafting impactful bullet points using the XYZ / STAR method.

The Solution

I developed a web application using Streamlit that acts as a targeted resume restructuring engine. Under the hood, it utilizes DeepSeek R1—chosen for its rigorous reasoning capabilities and cost efficiency—orchestrated via LangChain.

Instead of simply generating text from scratch, the system takes the user's raw, unformatted job descriptions and explicitly reconstructs them. The prompt pipeline forces the LLM to identify the implied Action, Context, and Result, outputting strictly formatted, quantifiable bullet points free of generic fluff.

DeepSeek R1

Utilized DeepSeek API for advanced NLP reasoning, instructing the model to strictly follow action-verb led, metric-driven formatting rules.

LangChain

Implemented LangChain for prompt templating and structured output parsing to ensure the LLM consistently returns actionable bullet points rather than paragraphs.

Streamlit UI

Built a rapid, interactive Python web front-end allowing users to paste raw experience or upload PDFs for instant restructuring.

The Outcome

The application successfully transforms scattered thoughts into professional, recruiter-ready bullet points in seconds. By chaining specific prompt engineering techniques, the tool avoids the "AI hallucination" trap, ensuring users maintain factual accuracy while massively improving the presentation of their impact.
