Flux Personal Finance Tracker

A modern, responsive budgeting application offering real-time expense tracking, dark mode aesthetics, and robust data visualizations.

Role

Full-Stack Developer

Timeline

Personal Project

Domain

FinTech, Web App UI/UX

Core Tech

React, Node.js, MongoDB

The Challenge

Many modern budgeting applications are either overly complex, requiring users to link bank accounts immediately, or too simplistic, lacking the granular categorization needed to actually understand spending habits. The goal was to build a secure, standalone financial dashboard that prioritizes manual entry speed, clear visual hierarchies, and a premium "glassmorphic" aesthetic that actually encourages users to log their daily expenses.

The Solution

I engineered Flux, a complete MERN-stack application (MongoDB, Express, React, Node.js). By adopting a component-driven architecture on the frontend, the UI remains consistently snappy whether the user is adding a single coffee receipt or filtering months of historical data.

The backend utilizes JWT-based authentication to secure personal financial data and exposes a robust REST API for the React client to consume. To visualize the data organically, I integrated dynamic charting libraries to break down spending categories into easily readable, interactive graphs.

State Management

Leveraged React Context and custom hooks to manage complex global state efficiently, ensuring that as new expenses are logged, all charts and summaries update instantaneously without page reloads.

Data Visualization

Implemented responsive, interactive charts (via Chart.js or Recharts) to provide users with immediate, actionable insights into their monthly cash flow and categorical spending.

Secure Backend Auth

Engineered a custom Node/Express backend that securely hashes passwords using bcrypt and protects API routes with JSON Web Tokens (JWT) to ensure strict data privacy.

The Outcome

Flux provides a seamless, distraction-free environment for users to take control of their finances. The responsive design ensures that the application is just as functional on a mobile device at the grocery store as it is on a desktop monitor during end-of-month reviews.
