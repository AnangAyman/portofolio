F1 Undercut Predictor Dashboard

A high-performance motorsport analytics tool visualizing real-time telemetry and undercut strategy viability in Formula 1.

Timeline

The Challenge

In Formula 1, the "undercut"—pitting before the car ahead to take advantage of fresh tires on an out-lap—is arguably the most powerful strategic maneuver on the track. However, assessing the viability of an undercut requires tracking multiple overlapping metrics: tire degradation curves, traffic in the pit window, pit lane loss time, and out-lap pace diffs. For the average viewer, this data is inaccessible or buried inside dense timing screens.

The Solution

I engineered a full-stack web application designed to ingest, process, and beautifully visualize real race data specifically centered around pit strategy. By hooking into the FastF1 Python library, I built a reliable data extraction pipeline that pulls Session timing, Lap timings, and Car Telemetry.

Instead of returning raw JSON to the front-end, the backend processes the delta times and calculates the precise "Undercut Threat Margin" between two selected drivers, serving this to a highly responsive Next.js frontend equipped with dynamic Recharts graphs.

FastF1 Pipeline

Engineered a robust Python backend that leverages Pandas to aggressively filter and cache large multi-lap telemetry payloads directly from F1's live timing API systems.

Delta Computations

Developed custom algorithms to normalize lap times, stripping out safety car anomalies and VSC phases to compute true racing pace and tire-degradation fall-off.

Next.js Dashboard

Designed a sleek, dark-mode React interface utilizing Recharts to instantly plot sector-by-sector driver comparisons and project the required margin to successfully undercut.

The Outcome

The resulting dashboard successfully translates thousands of overwhelming data points into an intuitive visual story. Users can select historic race sessions and immediately quantify why specific strategies succeeded or failed, providing broadcast-level strategic insights directly in their browser.
